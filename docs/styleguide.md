# RouteIQ App Style Guide (Finalized)

This guide defines the core visual elements for the RouteIQ application and related static/marketing pages, ensuring a consistent user experience.

## 1. **Color Palette**

### Primary & Accent Colors

| Name    | Hex Code | Usage    |
|---|---|---|
| **Accent Red** | #F44336 (Approx.) | **Primary Call-to-Action (CTA)** buttons, active navigation states, key status indicators (e.g., the active 'Week' button). |
| **Primary Text (Dark)** | #1A1A1A (Approx.) | Main headings, body text, high-contrast areas. |
| **Secondary Text (Grey)** | #666666 (Approx.) | Subheadings, descriptive text, metadata (e.g., "Welcome back! Here's your activity overview."). |
| **Background (Light)** | #FFFFFF | Main page background, card backgrounds. |
| **Subtle Highlight (Pinkish)** | #FFFFF0 (Approx.) | Used for highlighting icons within the data cards. |
| **Border/ Divider (Light Grey)** | #E0E0E0 (Approx.) | Subtle borders on cards and dividers. |

## 2. Typography

Major titles use the distinct **Orbitron** font. All other text uses a clean, highly legible **Sans-Serif** typeface (e.g., Inter, Roboto).

## 3. Components

### Navigation Bar (Header)

- **Logo/Brand Name:** Top left, **Primary Text** color (potentially using Orbitron).
- **Main Navigation Links:** Regular weight, Secondary Text.
  - Active Link: Highlighted with **Accent Red** text and an underline.
- **Utility Links:** Smaller font, typically 'Demo', 'free', 'Logout'.

### Data Cards (Metrics)

- **Structure:** Clean, white background, square corners, minimal bordering.
- **Content:**
  - Icon: Contained within a small square with a **Subtle Highlight (Pinkish)** background.
  - Metric Value (Large, Bold, Primary Text).

### Button & State Controls

- **Primary Button (Active State):** Solid block of **Accent Red**, white text (e.g., Week).
- **Secondary/inactive Button:** White background, **Primary Text** or **Secondary Text** border (e.g., Month, Year, All).

### Lists (Recent Activities)

- **Divider:** A thin, subtle **Border/Divider** line separates entries.
- **Metadata Layout:** Information (distance, time, pace) is aligned right using **Secondary Text** for quick scannability.
